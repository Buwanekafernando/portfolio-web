---
description: how to run the portfolio project
---

To run the project locally, follow these steps:

1.  Open a terminal in the `portfolio` directory:
    ```bash
    cd portfolio
    ```
2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```
3.  Start the development server:
    // turbo
    ```bash
    npm start
    ```
4.  Open your browser to [http://localhost:3000](http://localhost:3000) to view the portfolio.
